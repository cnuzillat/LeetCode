# LeetCode Solution
Welcome to my LeetCode Solutions repository!
This repo contains my personal solutions to LeetCode problems, organized by problem number. Each solution aims to be clean, efficient, and well-explained.

## Goals
- Strengthen problem-solving and algorithmic thinking
- Prepare for coding interviews
- Build a well-structured reference of solved problems
- Track progress over time

## What’s Included
- Solutions in languages I practice (mainly Python)
- Clean and readable code
- Time and space complexity notes
- Occasional multiple approaches (brute force → optimal)
- Comments explaining logic where helpful

## How to Use This Repo
- Search by the LeetCode problem number
- Compare different solution approaches
- Use as a reference for interview preparation
